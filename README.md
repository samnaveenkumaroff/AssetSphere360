# AssetSphere 360

Enterprise Inventory & ERP System

## Stack
- **Backend:** ASP.NET Core 10 · C# · Clean Architecture
- **Frontend:** Angular 21 · TypeScript · Angular Material
- **Database:** MS SQL Server 2022
- **Cache:** Redis 7
- **ORM:** Entity Framework Core 10
- **OS:** Ubuntu 24.04 LTS
